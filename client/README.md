# Premium Boutique Frontend

A premium, responsive React frontend built with Vite, utilizing modern design tokens and glassmorphism.

## Prerequisites
- [Node.js](https://nodejs.org/) (Version 14 or higher)

## Setup & Installation

1. **Install Dependencies**:
   ```bash
   npm install
   ```

## Running the Application

Start the Vite development server:
```bash
npm run dev
```
The application will be accessible at the URL shown in your terminal (usually `http://localhost:5173`).

## Key Features
- **Responsive Design**: Works beautifully on mobile, tablet, and desktop.
- **Identity Flow**: Automatic user tracking via Device ID.
- **Glassmorphism Sidebar**: Sleek cart experience.
- **Invoice Generation**: Download PDF invoices immediately after purchase.

## Tech Stack
- **React** (v19)
- **Vite** (Build Tool)
- **Lucide React** (Premium Icons)
- **jsPDF** (Invoice Generation)
- **Axios** (API Requests)
- **Vanilla CSS** (Custom Design System)
